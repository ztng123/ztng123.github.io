---

&nbsp;

### ❗ERROR! The server quit without updating PID file

&nbsp;

서버가 PID 파일을 업데이트하지 못하고 강제로 종료되었거나 PID 파일을 생성할 수 없는 상태

&nbsp;

👉 디렉토리의 소유권과 파일에 대한 권한이 없어서 생긴 오류

&nbsp;

&nbsp;


## 📂 MySQL 데이터 디렉토리의 소유권 변경

&nbsp;

```
sudo chown -R mysql:mysql /디렉토리 경로
```

&nbsp;

시도했으나 해결되지 않았다

&nbsp;

👉 var 폴더가 없었다

&nbsp;

&nbsp;


## 🗑 제거 후 다시 재설치

&nbsp;

```
brew install mysql
```

&nbsp;

다시 재설치 했더니 해결되었다 !
